# HOW TO POST TO GITHUB

## Step 1: Create Repository on GitHub Website

1. Go to: https://github.com/new
2. Fill in:
   - **Repository name**: `personala-dashboard`
   - **Description**: "A comprehensive, privacy-first AI assistant that runs entirely on your local machine"
   - **Public** (radio button)
   - **DO NOT** check any boxes
3. Click: **Create repository**

## Step 2: Open Command Prompt

Press `Windows Key + R`, type `cmd`, press Enter

## Step 3: Navigate to Your Project

Type this command and press Enter:
```
cd C:\ai_dashboard
```

## Step 4: Set Up Git (One Time Only)

Type these commands one by one:
```
git config user.name "norhtecmbarnes"
git config user.email "your-email@example.com"
```

## Step 5: Add All Files

Type:
```
git add .
```

## Step 6: Commit Files

Type:
```
git commit -m "Initial release: PersonalAI Dashboard v0.1.0"
```

## Step 7: Connect to GitHub

Type:
```
git remote add origin https://github.com/norhtecmbarnes/personala-dashboard.git
```

## Step 8: Push to GitHub

Type:
```
git push -u origin master
```

## Step 9: Verify

Go to: https://github.com/norhtecmbarnes/personala-dashboard

You should see your code!

---

## If You Get Errors:

**Error: "fatal: not a git repository"**
- Run: `git init` first, then try again

**Error: "Permission denied"**
- Make sure you're logged into GitHub in your browser
- Check that the repository name is exactly: `personala-dashboard`

**Error: "failed to push"**
- Run: `git pull origin master` first, then try push again
